# 7. Reverse integer

Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the signed 32-bit integer range [powe(-2, 31), pow(2,31) - 1], then return 0.

### Example 1:
* Input: x = 123
* Output: 321

### Example 2:
* Input: x = -123
* Output: -321

### Example 3:
* Input: x = 120
* Output: 21

### Constraints
* pow(-2, 31) <= x <= pow(2, 31) - 1
